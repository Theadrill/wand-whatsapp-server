# wand-whatsapp-server
Servidor WebSocket construído com Node.js puro e Baileys para espelhamento inteligente de notificações do WhatsApp. O W.A.N.D. gerencia filtros (Whitelist/Blacklist), histórico local (SQLite) e download de mídia sob demanda para alimentar clientes desktop/mobile conectados via rede local ou túneis como Tailscale.
